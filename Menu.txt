[[DEVICE_SETUP]]
	[[WIFI]] 
		-B Сканирование сети
		-E SSID
		-E пароль точки доступа
		-V DHCP
		- статический IP
			- IP
			- маска
			- шлюз
		
		- точка доступа по умолчанию
		
		-B Сохранить
	
	[[GSM]] 	
		- тестовая отправка СМС
		- шаблоны СМС
		- номер ТА для отправки смс
		- прием СМС для аварийной остановке
		-V контроль внешнего питания
			- СМС об отключении питании
		
	[[MQTT]]
		-V Включить
		-E IP сервера
		-E Порт
		-E Имя пользователя
		-E Пароль
		-E Заголовок
		-E Интервал обновления
		-V Автоматическое обнаружение
		-B Сохранить
		
	[[BOILER_TEMPERATURE]]
		- погрешность температуры воды в бойлере (+-1)
		- температура воды в бойлере (60С)
		- температура теплоносителя (90С)
		- максимальная температура бойлера(аварийная)
		- температура бойлера по времени (ночной режим)
		
	[[HEATING_TEMPERATURE]]
		-V режим зима-лето
			- 
		- погрешность температуры теплоносителя (+-3)
		- температура теплоносителя (55С) 
		- регулировка теплоносителя от внешней температуры
		- время работы насоса после прогрева отопления (0 - работа на постоянку)
	
	[[TESTING]]
		- тест всех реле
	
[[STATISTICS]]
	[[NETWORK_STATUS]]
		- состояние WIFI, GSM, MQTT
		
	[[STATE_SYSTEM]]
		- Основные
			- температура помещения где стоит устройство
			- время работы
			- ESP температура : 32.00 °C
			- Версия прошивки : 0.1.3
			- Оборудование : WT32-ETH01
			- ESP32 модель : ESP32-D0WDQ5
			- Процессор : 2 cores @ 240 MHz
			- Флеш-память : 4 Mb, внешняя
			- Свободная ОЗУ : 222 / 300 KiB
		
		
	[[CONSOLE]]
		- консоль логов
			- очистка
